
# 因为忘记早睡而晚睡

- 这可能是很常见的情况。
- 因为我们被其他东西吸引、拖延、耽误而让我们直接忘记了早睡。
- 或是间断性的想起要早睡，但想法又一下子消散。又去做其他事项了。
- 最后想要睡觉时，已经很晚了，并且感到后悔和焦虑。

- 所以我们或许需要得到提醒、或是温柔的监督。（或许不温柔的监督也不错？）
- 总之通过各种方法，来让我们想起、并去早睡。
  - 打破其他东西对我们的干扰，把早睡这一事项清晰的放在我们眼前。
